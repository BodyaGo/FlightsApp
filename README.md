# FlightsApp

Simple FlightsApp

## Installation

To install and set up the project, follow these steps:

   Clone the repository:

   ```bash
   git clone https://github.com/BodyaGo/FlightsApp.git
   ```
Navigate to the project directory:
```bash
   cd client
```
Install the dependencies:
```bash
   npm install
   or
   yarn install
 ```
Start the project:
```bash
   npm start
   or
   yarn start
 ```
