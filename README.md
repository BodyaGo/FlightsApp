# FlightsApp

Simple FlightsApp

## Installation

To install and set up the project, follow these steps:

   Clone the repository:

   ```bash
   git clone https://github.com/BodyaGo/FlightsApp.git
 ```
Navigate to the project directory:

   cd client

Install the dependencies:

   npm install
   or
   yarn install

Start the project:

   npm start
   or
   yarn start
